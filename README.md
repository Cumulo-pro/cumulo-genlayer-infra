# GenLayer Validator Node — Operational Documentation

This repository contains Cumulo’s operational documentation for running a GenLayer validator node in testnet.

The documentation includes:

- Installation guide
- Directory structure
- Wallet & keystore handling
- Systemd configuration
- Monitoring essentials
- Common commands & troubleshooting
- Full FAQ

> These documents are generic and can be adapted to any validator infrastructure.

## Files
- `INSTALL.md` — Installation & setup
- `OPERATIONS.md` — Daily operations, commands, logs, systemd usage
- `FAQ.md` — Common errors, how to fix issues, best practices

For questions or improvements, contact the Cumulo infrastructure team.
